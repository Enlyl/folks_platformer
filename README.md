## English

### 🎮 About

**Punk Platformer** is a 2D side-scrolling platformer game built with pure HTML5 Canvas and JavaScript. No external dependencies required — just open `index.html` in your browser and play!

### Features / Особенности

- **Character Customization** — Create your unique punk with customizable hair, shirts, patterns, and pants
- **Guitar Shop** — Collect coins, defeat enemies, and unlock awesome guitars
- **Dynamic Resolution** — Supports multiple resolutions from 1280x720 to 2560x1440
- **Procedural Graphics** — All visuals are drawn programmatically (no external assets needed)
- **Gamepad Support** — Play with keyboard or a gamepad
- **Multiple Enemies** — Angry robots patrol the platforms

### Controls / Управление

| Action | Keyboard | Gamepad |
|--------|----------|---------|
| Move | ← → or A/D | Left Stick |
| Jump | Space / Up / W | A Button |
| Attack | E | X Button |
| Menu Select | Enter / E | A Button |

### How to Play / Как играть

1. Open `index.html` in any modern browser
2. Customize your character on the start screen
3. Press **Play** or **Space** to start
4. Collect coins, avoid enemies, reach the flag!

### Resolution Settings / Настройки разрешения

Go to **Settings** in the main menu to change resolution:
- 1280 x 720
- 1366 x 768
- 1600 x 900
- 1920 x 1080 (default)
- 2560 x 1440

### Technical Details / Технические детали

- **Single HTML file** — All code, styles, and graphics in one file
- **~3000 lines** of JavaScript
- **Web Audio API** for sound effects
- **RequestAnimationFrame** for smooth 60fps gameplay
- **No external dependencies**

### Browser Support / Поддержка браузеров

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+

## Русский

### 🎮 О проекте

**Панк Платформер** — это 2D сайд-скроллинговый платформер, созданный на чистом HTML5 Canvas и JavaScript. Никаких внешних зависимостей — просто откройте `index.html` в браузере и играйте!

### Особенности

- **Кастомизация персонажа** — Создайте своего уникального панка: причёска, майка, узор, штаны
- **Магазин гитар** — Собирайте монеты, побеждайте врагов и покупайте крутые гитары
- **Динамическое разрешение** — Поддержка разрешений от 1280x720 до 2560x1440
- **Процедурная графика** — Вся графика рисуется программно (никаких внешних ассетов)
- **Поддержка геймпада** — Играйте с клавиатуры или геймпада
- **Несколько видов врагов** — Злые роботы патрулируют платформы

### Управление

| Действие | Клавиатура | Геймпад |
|---------|-----------|---------|
| Движение | ← → или A/D | Левый стик |
| Прыжок | Пробел / ↑ / W | Кнопка A |
| Атака | E | Кнопка X |
| Выбор в меню | Enter / E | Кнопка A |

### Как играть

1. Откройте `index.html` в любом современном браузере
2. Настройте своего персонажа на стартовом экране
3. Нажмите **Играть** или **Пробел**
4. Собирайте монеты, избегайте врагов, доберитесь до флага!

### Настройки разрешения

Зайдите в **Настройки** в главном меню, чтобы изменить разрешение:
- 1280 x 720
- 1366 x 768
- 1600 x 900
- 1920 x 1080 (по умолчанию)
- 2560 x 1440

### Технические детали

- **Один HTML-файл** — Весь код, стили и графика в одном файле
- **~3000 строк** JavaScript
- **Web Audio API** для звуковых эффектов
- **RequestAnimationFrame** для плавного геймплея 60fps
- **Без внешних зависимостей**

### Поддержка браузеров

- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+
